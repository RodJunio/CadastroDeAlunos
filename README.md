# CadastroDeAlunos
Realizado em Asp.Net MVC realizei um CRUD de cadastro de alunos.

Aplicação em Asp.NET MVC para cadastrar alunos. Além do cadastro é possível alterar e deletar alunos.

Os cadastros ficam salvos no banco de dados (utilizado SQL Server).
